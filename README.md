   * 命令行参数说明

     一个命令参数：
     
     | 参数 | 功能                    |
     | ---- | ----------------------- |
     | show | 显示已创建的connection  |
     | help | 显示参数使用帮助        |
     | ssid | 显示当前环境的wifi ssid |
     
     3个参数
     
     | 参数      | 功能           |
     | --------- | -------------- |
     | con_name  | 创建connection |
     | wifi_ssid | 需要连接的ssid |
     | password  | ssid的密码     |
